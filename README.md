##  Overview
This repository contains **combinational digital logic projects** implemented in **Verilog HDL**  
and verified in **Xilinx Vivado**. The projects focus on **basic combinational circuits and arithmetic logic designs**.

##  Platform & Tools
- **HDL:** Verilog  
- **EDA Tool:** Xilinx Vivado  
- **Simulation:** Vivado Simulator
- 
##  Projects Included

###  Logic Gates & Basic Circuits
- AND, OR, NOT, NAND, NOR, XOR, XNOR  
- Multiplexers (2:1, 4:1)  
- Demultiplexers  

###  Arithmetic & Combinational Modules
- Half Adder, Full Adder  
- Half Subtractor, Full Subtractor  
- 4-bit Adder/Subtractor  
- Comparator Circuits  

### Other Modules
- Encoder / Decoder Circuits  
- Priority Encoder
